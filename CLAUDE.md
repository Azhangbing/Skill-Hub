# Skill Hub 平台

厦门雅迅智联科技股份有限公司 - 内部技能管理平台

## 技术栈
- 前端：React + Vite + Tailwind CSS
- 后端：Node.js + Express
- 数据库：MySQL (skill_hub)
- 认证：JWT

## 启动服务

```bash
# 后端 (端口 8080)
cd backend && npm start

# 前端 (端口 3000)
cd frontend && npm run dev
```
