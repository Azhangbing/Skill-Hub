# 🚀 Ubuntu 环境下配置 Claude Code 全指南

本文档详细介绍如何在 Ubuntu 系统上完成 Claude Code 的完整配置流程。

> [!tip] 快速验证
> 配置完成后，在终端输入 `/` 检查是否出现 `uts_coding` 等 Skill 提示。
>
> ![[Pasted image 20260211135024.png]]

---

## 1. 环境准备

### 1.1 升级 Node.js

Claude Code 要求 Node.js **v18+** 版本，推荐使用 `nvm` 管理：

```bash
# 安装 npm
sudo apt install npm -y

# 安装 NVM (使用镜像加速)
curl -o- https://cdn.jsdelivr.net/gh/nvm-sh/nvm@v0.39.1/install.sh | bash

# 重新加载配置
source ~/.bashrc

# 安装并使用最新 LTS 版本
nvm install --lts
nvm use --lts

# 验证版本
node -v
```

> [!success] 版本要求
> 确保 Node.js 版本 >= 18.0.0

---

## 2. 安装 Claude Code

### 2.1 安装核心工具

使用国内镜像源加速安装：

```bash
# 安装 Claude Code
npm install -g @anthropic-ai/claude-code@latest --registry=https://registry.npmmirror.com

# 安装消耗统计工具 (可选)
npm install -g ccusage --registry=https://registry.npmmirror.com
```

> [!info] 镜像说明
> 使用 `registry.npmmirror.com` 可显著提升国内下载速度。

---

## 3. 核心配置

### 3.1 创建配置文件

```bash
mkdir -p ~/.claude
touch ~/.claude/settings.json
```

### 3.2 编辑配置内容

在 `~/.claude/settings.json` 中添加以下配置：

```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "your-key-here",
    "ANTHROPIC_BASE_URL": "https://fast.poloai.top",
    "CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1"
  },
  "diffTool": "auto",
  "tools": ["Bash", "Read", "Edit", "Grep", "Glob"],
  "permissions": {
    "allow": [],
    "deny": []
  }
}
```

> [!warning] 重要配置项
> - `ANTHROPIC_AUTH_TOKEN`: 你的 API 密钥
> - `ANTHROPIC_BASE_URL`: API 代理地址（国内访问必需）
> - `tools`: 启用的工具列表

---

## 4. 常见问题

### 4.1 首次执行失败（地域限制）

> [!bug] 错误提示
> `Unable to connect to Anthropic services...`

**解决方案**：

编辑 `~/.claude.json`，在最外层大括号内添加：

```json
"hasCompletedOnboarding": true
```

完整示例：

```json
{
  "hasCompletedOnboarding": true,
  "otherSettings": "..."
}
```

---

## 5. 验证安装

启动 Claude Code 并测试：

```bash
claude
```

在交互界面中输入 `/`，检查是否显示可用的 Skills。

> [!success] 配置成功标志
> - 能够正常启动 Claude Code
> - 输入 `/` 显示 Skill 列表
> - 可以正常对话和执行命令
