## 第1步 · 安装 Claudian 插件

### 📥 下载和安装步骤

> [!abstract] 准备工作
> 从 [GitHub 最新版本](https://github.com/YishenTu/claudian/releases/latest) 下载以下文件：
> - `main.js`
> - `manifest.json`
> - `styles.css`

**安装流程：**

1. **创建插件目录**
   ```
   /path/to/vault/.obsidian/plugins/claudian/
   ```
> [!warning] - "/path/to/vault"为你知识库所在根目录
2. **复制文件**
   - 将下载的 3 个文件复制到上述目录

3. **启用插件**
   - 打开 Obsidian 设置
   - 导航到 **设置 → 社区插件**
   - 搜索并启用 **"Claudian"**

## 第3步 · 选择 AI 模型配置

### 🤖 Claude Code（推荐）

> [!success] 推荐配置
> 使用 Claude Code 以获得最佳体验

**环境变量配置：**

```env
ANTHROPIC_AUTH_TOKEN=your-api-key
ANTHROPIC_BASE_URL=https://fast.poloai.top
API_TIMEOUT_MS=3000000
CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1
```

**配置位置：**
- Windows: `%APPDATA%\.claude\` 或 `~/.claude/`
- Mac/Linux: `~/.claude/`

## 第4步 · 安装 Skills（技能扩展）

### 💡 Obsidian Skills 概述

> [!abstract] Skills 简介
> **Obsidian Skills** 是为 Obsidian 设计的扩展技能包
>
> 📚 官方仓库：https://github.com/kepano/obsidian-skills
>
> 这些技能遵循 [Agent Skills 规范](https://agentskills.io/specification)，因此任何兼容技能的代理都可以使用它们，包括 **Claude Code** 和 **Codex CLI**。

---

### 📦 Skills 安装方式

#### 方式 1️⃣ ：通过 Plugin Marketplace（推荐）

```bash
/plugin marketplace add kepano/obsidian-skills
/plugin install obsidian@obsidian-skills
```

#### 方式 2️⃣ ：使用 npx

```bash
npx skills add git@github.com:kepano/obsidian-skills.git
```

#### 方式 3️⃣ ：手动安装

**📌 对于 Claude Code：**

将仓库内容添加到 `/.claude` 文件夹：
```
/.claude/
  └── obsidian-skills/
      └── skills/
```

更多信息见 [Claude Skills 官方文档](https://platform.claude.com/docs/en/agents-and-tools/agent-skills/overview)

**📌 对于 Codex CLI：**

复制 `skills/` 目录到：
```
~/.codex/skills/
或
/etc/skills/
```

参考 [特工技能规范](https://agentskills.io/specification)

**📌 对于 OpenCode：**

克隆整个仓库到 OpenCode 技能目录：
```bash
git clone https://github.com/kepano/obsidian-skills.git ~/.opencode/skills/obsidian-skills
```

⚠️ **重要**：克隆整个仓库，不要只复制 `skills/` 文件夹，目录结构应为：
```
~/.opencode/skills/obsidian-skills/skills/<skill-name>/SKILL.md
```

OpenCode 会自动发现所有 `SKILL.md` 文件，重启后生效。

> [!success] 官方 Skills

| 技能名                   | 功能描述                                                 |

| **defuddle**          | 使用 Defuddle 从网页提取干净的 Markdown，去除冗余信息，节省 Token        |

## 🎓 快速开始

### ✅ 验证安装

1. **检查 Claudian 插件**
   - 打开 Obsidian 设置 → 已安装的插件
   - 确认 "Claudian" 已启用

2. **验证 API 配置**
   - 打开 Claude Code 命令面板
   - 测试是否能正常连接

3. **测试 Skills**
   - 在 Claude Code 中尝试使用 `/obsidian-markdown` 等 Skills
   - 确认能正常调用

## 💡 常见问题

> [!warning] 插件未出现
> 如果插件未在 Obsidian 中显示，检查：
> - ✓ 文件是否正确复制到 `.obsidian/plugins/claudian/` 目录
> - ✓ 是否有 `manifest.json` 文件
> - ✓ 是否需要重启 Obsidian

> [!warning] API Key 错误
> 如果出现 API Key 错误：
> - ✓ 检查 API Key 是否正确（无多余空格）
> - ✓ 确认 Token 未过期
> - ✓ 验证网络连接是否正常

> [!warning] Skills 无法使用
> 如果 Skills 无法调用：
> - ✓ 确认 Skills 文件已正确安装
> - ✓ 重启 Claude Code
> - ✓ 检查配置文件语法
